MCU - 產業

2026-06-04

待查證

常見有三種商業模式：

  • IP 授權： 不賣晶片，賣設計區塊（CPU core、介面 IP），讓別人放進自己的 SoC，這是上游的模式。
  • 標準品 MCU： 設計一顆通用晶片大量量產、賣給所有人，賣的是可量產的晶片平台 + 生態工具鏈，多數品牌廠走這條。
  • 客製 ASIC / 設計服務： 依客戶規格做成專屬晶片，賣的是設計能力與量產導入。

另外常聽到的 IDM 與 fabless，是「自己有沒有晶圓廠」的製造分法，和上面的販售模式不同。

供應鏈

把一顆 MCU 從技術到出貨拆開，大致是：

上游（IP / 工具）：Arm、RISC-V 架構授權、EDA 工具、矽智財
        ↓
MCU 設計公司（多為 fabless）：做標準品 MCU 或客製 ASIC
        ↓
代工（製造）：晶圓代工 foundry ＋ 封裝測試 OSAT
        ↓
客戶端：ODM / 控制板廠 / 模組廠 / 方案商 / 設備製造商
        ↓
終端品牌與消費者
  • 上游：多數 MCU 的 CPU 核心來自 Arm 的 Cortex-M 授權，近年 RISC-V 興起成為另一條路；再加上 EDA 工具與各種矽智財。
  • 代工：fabless 的 MCU 廠把晶圓交給代工廠、封測交給 OSAT，製程多落在成熟節點而非最先進製程。
  • 客戶端：MCU 很少直接賣進大品牌總部，多半先到 ODM、控制板廠、模組廠、方案商與設備製造商，再組進終端產品。

台廠

台灣在這條鏈幾乎每一段都有角色：

  • 代工：台積電、聯電是全球 MCU 重要的晶圓來源。
  • IP / ASIC 服務：智原（聯電集團）、創意、世芯，賣的是設計能力與矽智財，而非標準品。
  • 自有品牌 MCU：新唐（Nuvoton）、盛群（Holtek）為主，另有松翰、義隆、凌通、偉詮等，多以 8-bit 或利基應用見長。

待查證：雅特力（Artery）與 GD32（兆易創新）其實是中國廠、不是台廠，常被和台廠一起討論只是因為同處 32-bit Arm MCU 市場。前一版把雅特力歸在「台廠」需要修正。

成本與國際因素

MCU 屬成熟製程，國際大廠多為 IDM，新進與中國廠則多走 fabless；產線分布上多落在 28nm 製程以上（很多還在 40 / 55 / 90 / 130nm）：

  • 台灣：台積電、聯電、力積電、世界先進（VIS）。
  • 中國：SMIC（中芯）、華虹、晶合（Nexchip）。
  • 其他：GlobalFoundries 等。

在 2024 年台灣成熟製程約 43–44%、中國約 31–34%；但在中國被封鎖先進製程後，把資本灌進成熟製程擴產殺價，預估 2027 年中國超車台灣。以 28nm 為例：台積電／聯電維持在 5,000 美元/片以上，SMIC 在 2025 年初一度把 28nm 從 2,500 砍到 1,500 美元/片（−40%）。

雖然中國以國家補貼與低價資本建立起結構性優勢，但殺價搶單加上 AI 需求拉升，反而把成熟製程產能塞滿，SMIC、華虹、晶合、VIS 在 2026 年又陸續漲價約 10%，低價搶單的優勢正在縮小。另一方面，美方又以價格扭曲為由對中國成熟製程啟動 Section 301 調查，讓單純的成本問題變成供應鏈風險題（關稅、Section 301、斷供）。

待查證：預估 2027 年中國超車台灣，以上報價與市占為 2025–2026 區間數字，會隨景氣與政策變動，引用前宜更新來源。

企業

進入 2026 年，MCU 市場除了既有的價格競爭，另一個焦點是「AI + 安全」的規格升級 —— edge AI、NPU、secure boot 成為新賣點；再加上前面提到的地緣供應鏈風險，構成現在選型的三條主軸。

STM32（ST，意法半導體）

標準級的 32-bit MCU，Cortex-M 全系列最完整：超低功耗 L 系列、主流 F / G 系列、高效能 H 系列，近期還有 AI 導向的 N 系列；ST 本身是 IDM。

  • 優點：生態最完整（CubeMX / HAL、範例、社群、第三方支援），選型最廣、文件成熟、長期供貨穩。
  • 缺點：單價偏高，在價格戰中成本居劣勢；熱門料曾在缺貨潮交期拉長，反而養出一整票相容替代。

NXP

車用與工業強項，i.MX RT「跨界 MCU」主打高效能，S32 車用平台布局深。

  • 優點：車規、功能安全與安全子系統強，高階 crossover 效能好。
  • 缺點：價格與入門門檻高，生態較 ST 分散，學習曲線陡。

Renesas

車用 MCU 全球前段班，產品線含 RA（Cortex-M）、RX（自有核心）、RL78（低功耗）。

  • 優點：車用與工控深耕、產品線廣、類比整合強。
  • 缺點：歷經多次併購，工具與生態較分裂，新手較不友善。

Microchip

8-bit 時代的 PIC / AVR 王者，32-bit 有 SAM（Arm）與 PIC32。

  • 優點：供貨與長期供應承諾佳、類比與混合訊號周邊豐富、低階市場黏著度高。
  • 缺點：32-bit 高階效能與生態不如 ST，MPLAB 工具體驗見仁見智。

Infineon（英飛凌）

車用與功率半導體龍頭，MCU 有 AURIX（車用安全）、PSoC（可程式類比 / 觸控）、XMC。

  • 優點：車規安全（AURIX 在動力 / 底盤強）、功率與感測整合佳、PSoC 彈性高。
  • 缺點：偏車用 / 工業專業領域，消費與 maker 較少用，價格高。

GD32（兆易創新，中國）

最早也最成功的 STM32 相容陣營，Cortex-M 全系列外還有自家 RISC-V（GD32V）。

  • 優點：高度相容 STM32、成本低、供貨改善，在中國的生態最成熟。
  • 缺點：相容但非完全等價（外設細節有差），高階與車規深度不如國際大廠，帶地緣風險。

雅特力 AT32（中國）

成立於 2016 年，是「標準品 MCU」模式的典型：自己做 Cortex-M4 / M0+ 的 AT32 系列，靠與 STM32 pin-to-pin 和軟體相容、更高時脈與更好的成本／MHz 比，切入原本用 STM32F4 的設計（例如 AT32F403A 與 STM32F407 同樣 LQFP64 封裝、同為 Cortex-M4F，但跑到 288 MHz），官方強調是「相容但獨立」，非 clone、也非直接 drop-in。製造採 fabless，晶圓與封測委外，並提供 IDE、範例、reference code 等工具與應用支援；客戶端多是工控設備廠、馬達／電機產品、車載周邊模組、家電 ODM/OEM、IoT 與 5G 周邊、方案與模組商。

  • 優點：與 STM32F4 相容、時脈更高、成本／MHz 佳。
  • 缺點：產品廣度與生態不如 ST / GD，車規與高階有限，供應鏈與具名客戶資訊不透明，帶地緣風險。

新唐 Nuvoton（台灣）

原華邦微控制器部門獨立而來，走 Cortex-M（M0 / M23 / M4）＋ 8051 雙線，布局工控、音訊與安全（TrustZone）。

  • 優點：台廠在地供應與技術服務、性價比、安全與工控應用支援。
  • 缺點：國際生態與品牌聲量不如 ST，高階效能線較少。

盛群 Holtek（台灣）

以 8-bit 家電 / 量測 / 周邊見長，32-bit 屬補強角色。

  • 優點：低階性價比高、家電與週邊應用支援深、供貨穩。
  • 缺點：32-bit 廣度與效能有限，國際市場聲量小。

樂鑫 ESP32（Espressif，中國）

Wi-Fi / BLE 無線 MCU 的代表，maker 與 IoT 市場極普及，新線轉向 RISC-V（C 系列）。

  • 優點：整合無線、便宜、社群與生態龐大（Arduino / ESP-IDF）。
  • 缺點：含無線功耗較高，即時控制與類比非強項，定位不在車規。

RP2040 / RP2350（Raspberry Pi）

maker 取向，RP2350 升級為雙核並可選 RISC-V。

  • 優點：便宜、PIO 可程式 I/O 獨特、文件與社群好。
  • 缺點：類比與周邊較陽春，偏 maker 而非工業 / 車規。

其他中國相容陣營（MM32、N32、HC32、APM32 等）多走「STM32 相容 + 成本」路線；WCH 的 CH32 部分相容，但主力 CH32V 是 RISC-V，要分開看。

待查證：各家優缺點為市場通論與定位歸納，非逐項實測；型號系列與最新動態（如 STM32N6、GD32 新線、RP2350）引用前宜再確認。
