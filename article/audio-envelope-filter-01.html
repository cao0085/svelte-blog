Plugin - Envelope Filter

2026-05-28

從 Mutron III / Neutron 理解「自動 wah」的類比電路原理

整體架構

Envelope filter（自動 wah）的核心是兩條平行路徑，透過光耦合器（LED + LDR）連接成一個閉迴路：

  • 音頻訊號路徑：吉他聲音經過放大、濾波後輸出
  • 控制路徑：偵測訊號音量，回過頭去改變濾波頻率
訊號
  → U1b 放大 → SVF 濾波 → 帶通輸出
  → R14 → U3b 精密整流（消除 0.6V 死區）
  → C9 平滑成 DC 包絡（τ ≈ 220ms）
  → U3a 驅動 LED（Up: 越大聲越亮 / Down: 越大聲越暗）
  → Rx 限流 → LED 發光
  → 光照射 LDR（R8, R9）
  → LDR 阻值改變 → SVF 截止頻率改變
  → 聲音音色跟著音量包絡自動變化

光耦的角色是把控制路徑的波形大小，轉成控制濾波器的參數 —— 動態越大 LED 越亮，濾波頻率掃得越高。這就是為什麼這類效果被稱為Auto wah，它把你的彈奏力度直接轉化為音色掃頻。

輸入放大器（U1b）

U1b 是一個反相放大器，非反相輸入（+）直接接地，訊號從反相輸入（−）進入。選擇反相放大器的原因是其增益公式 Gain = -Rf/Rin 允許增益小於 1，可以同時處理過弱（放大）和過強（縮小）的訊號；非反相架構的增益最小為 1，無法縮小訊號。

這一級的作用是把各種不同輸出強度的訊號正規化到一個適合後級濾波器工作的範圍，U1b 的輸出同時送往兩個地方：State Variable Filter（做頻率濾波）與 Bypass 輸出（透過開關直接送到輸出 jack）。

增益怎麼算

訊號先經過 R1（3.3K）再串聯 GAIN 電位器（1M 反向對數），兩者相加是輸入電阻 R_in；回授電阻是 R3（120K）。

G = −R3 / (R1 + R_GAIN)
  • 最大增益：電位器趨近 0，R_in ≈ 3.3K，增益約 −36 倍（+31dB），足以把很弱的單線圈拾音器訊號推到飽滿的工作範圍
  • 最小增益：電位器加到接近 1M，R_in ≈ 1M，增益只有約 −0.12 倍（−18dB），等於大幅衰減，適合輸出很大的主動式拾音器或 humbucker

為什麼用反向對數電位器

人耳感知音量是對數的。反向對數的旋轉曲線讓旋鈕的「感覺」比較線性——轉一點點不會突然從很小跳到很大，增益調節比較好操控。

反相帶來的問題

因為是反相放大器，輸出和輸入永遠差 180°。Bypass 訊號也從這裡取出，所以 bypass 也是反相的。正常使用下（只用 effect 或只用 bypass）沒問題，但若要做平行混音（乾濕訊號混在一起），兩者會部分抵消。解決方案：

  • 採用跟 Mutron 原版一樣的緩衝旁路（buffered bypass）
  • 或用 DPDT 開關做 true bypass
  • 或用 Millennium Bypass 方案

R1 的額外作用

R1（3.3K）不只是增益公式的一部分，它也提供了一個最低輸入阻抗的保障。即使 GAIN 電位器轉到 0，輸入端阻抗不會低於 3.3K，避免對拾音器造成過度負載。不過 3.3K 對被動拾音器來說偏低，高頻會被稍微吃掉一些——這也是 Mutron 音色的一部分特徵。

State Variable Filter（U1a、U2b、U2a）

這是整個效果器的核心。大多數效果器用簡單的 RC 濾波器或 Sallen-Key 拓撲，但 Mutron III / Neutron 選擇 state variable filter，原因有三：

  • 一個電路三種輸出：高通、帶通、低通同時存在，用開關切換就好
  • 頻率和 Q 值可獨立控制：頻率靠 R×C 時間常數，Q 值靠回授量，互不干擾
  • 頻率控制只需要改電阻：用 LDR 就能掃頻，不需要可變電容

三個 op-amp 各做什麼

U1a — 加法/差分放大器：濾波器的入口，接收三個訊號做加權相減：

  • 從 U1b 來的輸入訊號（經 R6 進入反相端）
  • 從 U2a 來的低通回授（經 R10 進入反相端）
  • 從 U2b 來的帶通 PEAK 回授（經 PEAK 電位器進入非反相端）

U1a 的輸出就是高通輸出。直覺上：輸入訊號減去低通成分 = 高通。

U2b — 第一個積分器：高通訊號經 R8（LDR）進入 U2b，回授路徑上是 C6（1.8nF），構成反相積分器：

輸出 = −1/(R8×C6) × ∫(輸入)dt

對高通做積分，數學上等於帶通，所以 U2b 的輸出就是帶通輸出。

U2a — 第二個積分器：帶通訊號再經 R9（LDR）進入 U2a，回授路徑上是 C8（1.8nF）。對帶通再積分一次 = 低通，所以 U2a 的輸出就是低通輸出。低通輸出透過 R10 回授到 U1a 的反相端，形成閉迴路。

截止頻率怎麼決定

f₀ = 1 / (2π × R_LDR × C)

兩個積分器使用相同電容值（C6 = C8 = 1.8nF）和匹配的 LDR（R8 = R9），所以截止頻率由 LDR 阻值決定。典型 LDR 範圍：

  • 暗態（LED 滅）：幾百 kΩ → f₀ 很低（幾十 Hz）
  • 亮態（LED 全亮）：幾百 Ω ~ 幾 kΩ → f₀ 很高（幾 kHz）

這就是為什麼彈一個強音會讓聲音從暗到亮「嘩」一下掃上去。

RANGE 開關

當 Range 設在「Low」時，C5（2.2nF）並聯到 C6、C7（2.2nF）並聯到 C8，總電容從 1.8nF 變成約 4nF，時間常數變大，整個可掃頻範圍往低頻移動約一個八度。這讓你可以選擇效果器在中高頻段還是中低頻段工作。

PEAK 控制 — 正回授與共振

PEAK 電位器（150K 對數）把帶通輸出的一部分送回 U1a 的非反相端，形成正回授：

  • PEAK = 0：沒有正回授，響應平坦寬廣，Q ≈ 0.5–1
  • PEAK 適中：帶通尖峰變陡，截止處有共振峰 → 經典 wah 音色
  • PEAK 接近最大：Q 極高，濾波器瀕臨自激，發出類似正弦波的嘯叫

帶通訊號在截止頻率處振幅最大，回授回去又增強了該頻率的成分，形成自我增強的共振。PEAK 旋鈕本質上控制的是這個閉迴路的增益裕度距離 1 有多近。

理論上兩個積分器的 R×C 要一致，濾波器的數學模型才準確。實務上 LDR 不匹配對聲音影響不大；若要做到最好，可在暗光和中等光照下量幾顆 LDR 的阻值，選最接近的一對。

包絡偵測迴路

這條路徑的任務是：偵測訊號有多大聲 → 轉換成 DC 電壓 → 驅動 LED → 透過光改變 LDR 阻值 → 控制濾波頻率。分成三個子電路。

第一段：精密整流器（U3b + D1 + D2）

普通矽二極體有約 0.6V 的正向導通電壓。吉他訊號經前級放大後可能只有幾十到幾百 mV，若直接用二極體整流，小訊號會完全被 0.6V 的門檻吃掉，包絡偵測器對輕彈完全沒反應。

精密整流的做法是把 D1、D2 放在 U3b 反相放大器的回授迴路裡。關鍵機制是 op-amp 的開迴路增益（約 100,000 倍）讓二極體的等效導通電壓降低到 0.6V / 100,000 ≈ 6µV，幾乎等於零，所以即使幾 mV 的訊號也能精確整流。D1、D2 的方向配合 R14/R15 構成半波整流。

元件角色：

  • R14（330Ω）：輸入電阻，設定整流器增益
  • R15（47K）：回授電阻，增益 = R15/R14 ≈ 142 倍，同時放大包絡訊號
  • D1：整流路徑
  • D2：箝位路徑，防止 op-amp 飽和（飽和恢復很慢，會拖慢響應速度）

第二段：包絡電容（C9 = 4.7µF）

整流器輸出的是半波脈衝——一連串隨音樂起伏的正向尖峰。C9 把這些脈衝平滑成緩慢變化的 DC 電壓，這就是「包絡」。

  • 充電（音符起始）：當新脈衝峰值高於 C9 上現有電壓時，電流通過 R15 對 C9 充電；充電時間常數較短，包絡能快速跟上 attack
  • 放電（音符衰減）：訊號變小時 C9 通過後級電阻緩慢放電，時間常數 τ ≈ R15 × C9 ≈ 47K × 4.7µF ≈ 220ms

這個 220ms 的放電時間決定了濾波器掃頻的「回落速度」。太快會聽到紋波（嗡嗡聲），太慢則掃頻感覺遲鈍。4.7µF 是 Mutron III 原始值，提供恰到好處的 envelope following 感。

第三段：LED 驅動器（U3a）

U3a 把 C9 上的 DC 包絡電壓轉換成流過 LED 的電流。Direction 開關（Up/Down）會物理上重新配線 U3a 的輸入端和回授路徑：

UP 模式 — 非反相跟隨器

包絡 DC → U3a 的 + 端
輸出直接回授到 − 端（unity gain follower）

包絡電壓越高 → U3a 輸出越高 → LED 越亮 → LDR 阻值越低 → 濾波頻率越高。音色效果：彈下去「嘩」往上掃，像踩下 wah 踏板。

DOWN 模式 — 反相器

包絡 DC → U3a 的 − 端（通過 R18）
+ 端接到一個 DC 偏壓點
R19 做回授，形成反相放大器

包絡電壓越高 → U3a 輸出越低 → LED 越暗 → LDR 阻值越高 → 濾波頻率越低。音色效果：彈下去「嘩」往下掃，然後在音符衰減時頻率回升。

Rx 電阻（560Ω ~ 4.7K）：串聯在 U3a 輸出和 LED 之間，限制 LED 最大電流，需根據使用的光耦選項調整：

  • 太小 → LED 電流太大 → LDR 阻值變化範圍太寬 → 掃頻幅度過大
  • 太大 → LED 電流不夠 → LDR 幾乎不動 → 沒有掃頻效果

建議從 560Ω 開始往上調，直到掃頻幅度適中。
