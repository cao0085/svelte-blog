Plugin - Envelope Filter - 01

2026-05-28

理解 Envelope Filter 類比電路概念

核心是兩條平行路徑，透過光耦合器（LED + LDR）連接；音頻路徑負責濾波輸出，控制路徑偵測音量包絡並驅動 LED，動態越大 LED 越亮，濾波頻率掃得越高。這也是Auto wah別稱的由來。

  • 訊號路徑：訊號經放大、濾波後輸出
  • 控制迴路：偵測訊號音量，動態調整濾波頻率
訊號
  → U1b 放大 → SVF 濾波 → 帶通輸出
  → R14 → U3b 精密整流（消除 0.6V 死區）
  → C9 平滑成 DC 包絡（τ ≈ 220ms）
  → U3a 驅動 LED（Up: 越大聲越亮 / Down: 越大聲越暗）
  → Rx 限流 → LED 發光
  → 光照射 LDR（R8, R9）
  → LDR 阻值改變 → SVF 截止頻率改變
  → 聲音音色跟著音量包絡自動變化

輸入放大器

作用是把不同輸出強度的訊號，正規化成適合後級濾波器工作的範圍，同時送往SVF、Bypass。選擇反相放大器U1b的原因是其增益公式 Gain = -Rf/Rin 允許增益小於 1，可以同時處理過弱（放大）和過強（縮小）的訊號；非反相架構的增益最小為 1 無法縮小訊號。

增益計算

訊號先經過 R1（3.3K）再串聯 GAIN 電位器（1M 反向對數），兩者相加是輸入電阻 R_in；回授電阻是 R3（120K）。

G = −R3 / (R1 + R_GAIN)
  • 最大增益：電位器趨近 0，R_in ≈ 3.3K，增益約 −36 倍（+31dB），足以把很弱的單線圈拾音器訊號推到飽滿的工作範圍
  • 最小增益：電位器加到接近 1M，R_in ≈ 1M，增益只有約 −0.12 倍（−18dB），等於大幅衰減，適合輸出很大的主動式拾音器或 humbucker
  • R1 的額外作用: 提供了一個最低輸入阻抗的保障，即使 GAIN 電位器轉到 0，輸入端阻抗不會低於 3.3K，然後高頻會被稍微吃掉一些。
  • 反向對數電位器: 反向對數的旋轉曲線讓旋鈕的「感覺」比較線性——轉一點點不會突然從很小跳到很大，增益調節比較好操控。

反相放大器輸出和輸入永遠差 180°，若要做平行混音（乾濕訊號混在一起），需要特別處理否則會有相位問題。

State Variable Filter

SVF 相較於 Sallen-Key 等傳統拓撲，頻率與 Q 值可獨立調整，且積分器串接的結構天生穩定，特性有：

  • 一個電路 N 種輸出：高通、帶通、低通可同時存在，用開關切換輸出點即可
  • 頻率和 Q 值獨立控制：頻率由 R×C 時間常數決定，Q 值由回授量控制，互不干擾
  • 頻率控制只需改電阻：用 LDR 即可掃頻，不需要可變電容

這邊以三個 op-amp 實現：

  • U1a (加法/差分放大器): 濾波器入口，對三個訊號( 輸入訊號、U2a 的低通回授、U2b 的帶通 Q 回授)做加權相減，輸出即為高通。
  • U2b (第一個積分器): 對上述高通訊號積分一次，數學上等於帶通，輸出即為帶通。
  • U2a (第二個積分器): 對上述帶通訊號再積分一次，得到低通。低通輸出透過回授到U1a形成閉迴路。

迴路拓撲 & 回授

U1a（加法器）──→ U2b（積分器）──→ U2a（積分器）
   ↑                                     │
   └──────────── R10 負回授 ←─────────────┘

如果把 R10 剪掉 A 站就是一個普通放大器；B 站做積分會變成一個很粗糙的低通；C 站再積分成更粗糙的低通，失去特性。所以讓三個 op-amp 串聯成一個環，訊號在環內循環流動，根據環上三個不同節點抽出來要的輸出：

訊號    → [A站: 減法] → [B站: 積分] → [C站: 再積分] → 回到A站（回授）
            ↓              ↓              ↓
          探針 HP        探針 BP        探針 LP
            ↓              ↓              ↓
            ─────────── 三選一開關 ──────────→ 輸出
  1. 時間點 1（第一個瞬間）：C 站還沒有東西可以回送，回授 ≈ 0。A 站的輸出 ≈ 原始輸入，系統剛啟動，還沒有歷史累積。
  2. 時間點 2 ~ 4：B 站和 C 站的積分器持續累積過去的訊號。C 站的低通回授越來越大，A 站輸出中的低頻成分被越扣越多。
  3. 時間點 5：A 站的輸出 = 輸入₅ − C 站累積的低頻歷史（包含 1~4 所有時間點的處理結果）。你在 A 站聽到的不是單純的輸入₅，而是被扣掉過去累積的低頻成分之後的結果——這就是為什麼它是高通。

從這張圖可以看到，截止頻率和 RANGE 控制的元件都在環路內部（改變積分速度），PEAK 則是從環路外部另拉一條正回授旁路（改變共振強度）。因為走的是不同路徑，頻率和 Q 值才能獨立控制。

基礎閉合

  • R10（負回授）：裝在 C→A 之間，把低通送回加法器做減法，讓環路閉合成為濾波器。

截止頻率

  • R8、R9：裝在 A→B 和 B→C 之間，是站與站之間的「閘門」。LED 越亮 → 阻值越低 → 積分越快 → 截止頻率越高。
  • C6、C8：裝在 U2b 和 U2a 各自的回授路徑上，和 R8/R9 一起決定積分時間常數。
f₀ = 1 / (2π × R_LDR × C)

兩個積分器通常使用相同電容值（C6 = C8 = 1.8nF）和匹配的 LDR（R8 = R9），所以截止頻率由 LDR 阻值決定。典型 LDR 範圍：

  • 暗態（LED 滅）：幾百 kΩ → f₀ 很低（幾十 Hz）
  • 亮態（LED 全亮）：幾百 Ω ~ 幾 kΩ → f₀ 很高（幾 kHz）

這就是為什麼動態大會讓聲音從暗到亮「嘩」一下掃上去。

理論上兩個積分器的 R×C 要一致，濾波器的數學模型才準確。實務上 LDR 不匹配對聲音影響不大；若要做到最好，可在暗光和中等光照下量幾顆 LDR 的阻值，選最接近的一對。

RANGE

  • C5、C7：並聯在 C6/C8 旁邊，開關 ON 時總電容變大，時間常數變大，頻率整體往下移。

當 Range 設在「Low」時總電容從 1.8nF 變成約 4nF，時間常數變大，整個可掃頻範圍往低頻移動約一個八度，藉此控制效果在中高頻段還是中低頻段工作。

PEAK pot

從 B 站拉一條額外的旁路到 A 站的非反相端（+），把帶通輸出的一部分送回 U1a 的非反相端，形成正回授，帶通訊號在截止頻率處振幅最大，回授回去又增強了該頻率的成分，形成自我增強的共振。PEAK 旋鈕本質上控制的是這個閉迴路的增益裕度距離 1 有多近。：

  • PEAK = 0：沒有正回授，響應平坦寬廣，Q ≈ 0.5–1
  • PEAK 適中：帶通尖峰變陡，截止處有共振峰 → 經典 wah 音色
  • PEAK 接近最大：Q 極高，濾波器瀕臨自激，發出類似正弦波的嘯叫

Output

  • Bypass / Effect 開關：決定輸出接的是 SVF Effect 開關選出的那一路，還是直接從 U1b 拉過來的乾聲（bypass）

  • Effect 開關（三選一）：從環上三個節點各拉一條線到同一個三段開關，撥到哪個位置就聽哪個節點。

包絡偵測迴路

目的是控制光改變 LDR 阻值調整「截止頻率」。

偵測訊號有多大聲 → 轉換成 DC 電壓 → 驅動 LED → 透過光改變 LDR 阻值 → 控制濾波頻率。

音頻訊號路徑：
輸入 → U1b → U1a → R8[LDR] → U2b → R9[LDR] → U2a → 輸出(閉迴)
                      ↑                  ↑
                      │                  │
                   這兩個 LDR 就是被控制的終點
                      ↑                  ↑
                      └───── LED 光 ─────┘

包絡控制路徑：
U2b 帶通輸出 → R14 → U3b 精密整流 → C9 平滑 → U3a 驅動 → Rx → LED 
                 ↑         ↑           ↑          ↑          ↑
               第一段     第一段      第二段      第三段     第三段

第一段：精密整流器（U3b + D1 + D2）

從帶通輸出取樣，將交流音頻訊號轉成單方向脈衝，供後級追蹤包絡。普通二極體有 0.6V 導通門檻，小訊號會被直接吃掉。精密整流將二極體放入 op-amp 回授迴路，利用高開迴路增益將等效門檻壓低至 6µV，使輕彈等小訊號也能完整整流。R14/R15 比值決定整體偵測靈敏度。

  • R14（330Ω）：輸入電阻，設定整流器增益
  • R15（47K）：回授電阻，增益 = R15/R14 ≈ 142 倍，同時放大包絡訊號
  • D1：整流路徑
  • D2：箝位路徑，防止 op-amp 飽和（飽和恢復很慢，會拖慢響應速度）

第二段：包絡電容（C9 = 4.7µF）

負責平滑與記憶，將整流輸出的半波脈衝轉成緩慢變化的 DC 電壓，也就是「包絡」，C9 的容值決定反應速度。

  • 充電（音符起始）：新脈衝峰值高於 C9 現有電壓時充電，時間常數較短，包絡能快速跟上 attack
  • 放電（音符衰減）：訊號變小時 C9 通過後級電阻緩慢放電，τ ≈ R15 × C9 ≈ 47K × 4.7µF ≈ 220ms

220ms 的放電時間決定濾波器的回落速度。太快會出現紋波，太慢則掃頻感覺遲鈍。4.7µF 為 Mutron III 原始值，提供適當的 envelope following 手感。

第三段：LED 驅動器（U3a）

將 C9 上的 DC 包絡電壓轉換成 LED 電流。Direction 開關（Up/Down）重新配線 U3a 的輸入與回授路徑，決定掃頻方向；Rx 限制 LED 最大電流，決定掃頻幅度。

  • UP 模式 — 非反相跟隨器

    • 包絡 DC → U3a 的 + 端
    • 輸出直接回授到 (−) 端（unity gain follower）
    • 包絡越高 → 輸出越高 → LED 越亮 → LDR 阻值越低 → 頻率越高。彈下去往上掃，像踩下 wah 踏板。

  • DOWN 模式 — 反相器

    • 包絡 DC → U3a 的 − 端（通過 R18）
    • (+) 端接 DC 偏壓，R19 做回授形成反相放大器
    • 包絡越高 → 輸出越低 → LED 越暗 → LDR 阻值越高 → 頻率越低。彈下去往下掃，音符衰減時頻率回升。

Rx 串聯在 U3a 輸出與 LED 之間建議從 560Ω 開始往上調，直到掃頻幅度適中。(太小 → 電流過大 → 掃頻幅度過寬，太大 → 電流不足 → 幾乎無掃頻效果)
