記憶體 - 產業

2026-06-05

待查證

接續 memory-01 的基礎,這篇談記憶體的產業結構——商業模式、供應鏈、客戶與主要企業。和 mcu-02 對照著看,會發現同樣是半導體,記憶體幾乎每一格都走相反路線。

商業模式

MCU 有 IP 授權、標準品、客製 ASIC 三種模式;記憶體則高度集中在少數幾種:

  • IDM 顆粒廠: 自己設計、自有晶圓廠製造,賣標準化的 DRAM / NAND 顆粒,這是主舞台,三星、SK 海力士、美光都是。
  • 主控 IC: 不做顆粒,做 NAND 旁邊那顆控制晶片與韌體,讓顆粒能組成 SSD,台廠群聯、慧榮走這條。
  • 模組 / 品牌: 買顆粒回來組成記憶體模組(DIMM)或 SSD、貼牌賣,賺的是組裝、通路與品牌,威剛、十銓、金士頓屬之。

和 mcu-02 最大的差別:MCU 可以 fabless 輕資產起家,記憶體顆粒這端幾乎只有重資本 IDM 玩得起,輕資產的機會被擠到主控與模組這兩個邊緣。

供應鏈

把一顆記憶體從技術到出貨拆開,大致是:

上游(設備 / 材料):EUV / 曝光機、矽晶圓、特化材料
        ↓
顆粒廠(IDM):三星、SK 海力士、美光、鎧俠 —— 設計＋製造一手包
        ↓
主控 + 模組:主控 IC(群聯、慧榮)＋ 模組 / SSD 廠
        ↓
客戶端:伺服器 ODM / 雲端 / PC / 手機 / 顯卡 / 車用
        ↓
終端品牌與消費者
  • 上游: 記憶體吃先進製程,所以設備(艾司摩爾 ASML、應材、東京威力科創)與材料是命脈,這點和邏輯晶片一樣。
  • 顆粒廠: 不像 MCU 把晶圓外包,IDM 自己設計又自己製造,這是記憶體的核心。
  • 客戶端: 大客戶常直供,中小客戶才走模組與通路(下一節展開)。

對照 mcu-02 的供應鏈圖,最大不同是中間少了「fabless 設計 → 代工 foundry → 封測 OSAT」這段分工——記憶體 IDM 把這幾段全收進自己手裡。

客戶

記憶體的客戶結構和 MCU 相反:MCU 多半要先繞過 ODM、模組廠才進終端,記憶體則因為量大、規格統一,大客戶常常直接買。

誰最終在用(按需求量):

  • 伺服器 / 資料中心: 現在最大宗,AI 把 DRAM(尤其 HBM)與企業級 SSD 的需求往上拉。
  • PC / 筆電: 戴爾、惠普、聯想等,用 DDR + SSD。
  • 智慧手機: 蘋果、三星、小米等,用 LPDDR + UFS。
  • 顯卡 / AI 加速器: Nvidia、AMD,用 GDDR 與 HBM。
  • 車用 / 工控 / 消費電子: 量分散,但車用對品質與供貨年限要求高。

怎麼賣到客戶手上,主要兩條路:

  • 直供大客戶: 伺服器 ODM、雲端巨頭(Google、Amazon、Meta)、手機品牌、Nvidia 等,量大到值得簽長約直接談。
  • 走模組 / 通路: 顆粒賣給模組廠組成 DIMM 或 SSD,再進品牌與零售,服務中小客戶與 DIY 市場。

HBM 是特殊的一種:不走通路,顆粒廠直接綁 AI 晶片廠(Nvidia、AMD),再由台積電 CoWoS 封進 GPU;因為供不應求,客戶要預付、長約鎖產能,關係從「買零件」變成「搶產能、結盟」。

台廠

對照 mcu-02 的台廠段:MCU 那邊台廠靠「自有品牌 + 利基應用」站穩;記憶體這邊台廠擠不進 DRAM / NAND 顆粒三強,但在幾個縫隙很穩:

  • 利基記憶體: 南亞科(利基型 DRAM)、華邦(NOR + 利基 DRAM)、旺宏(NOR / ROM,車用與消費)。
  • NAND 主控: 群聯、慧榮,靠控制晶片與韌體切入,不碰顆粒。
  • 模組 / 通路: 威剛、十銓,買顆粒組裝貼牌。

簡單說,顆粒製造這條台廠打不過韓美日的重資本 IDM,但 NOR、利基、主控、模組這些「需要彈性與韌體、不需追最尖端製程」的位置,反而是台廠的主場。

企業

三星 Samsung(韓,IDM)

DRAM 與 NAND 雙料龍頭,也是全產業垂直整合最廣的公司——記憶體、晶圓代工、自研晶片、面板、終端產品全包(詳見和台積電的對照)。

  • 優點: 規模最大、製程領先、垂直整合能自我供應,財力雄厚抗週期。
  • 缺點: 既做代工又做終端,和客戶有競爭關係;HBM 一度落後 SK 海力士。

SK 海力士 SK Hynix(韓,IDM)

DRAM 強、NAND 靠併購 Solidigm 補齊。近年最大亮點是 HBM——在 AI 記憶體這塊一度領先,綁定 Nvidia。

  • 優點: HBM 領先、與 AI 龍頭結盟、DRAM 技術強。
  • 缺點: 產品廣度與財力不如三星,NAND 整合仍在消化。

美光 Micron(美,IDM)

三強中唯一的美系廠,DRAM / NAND 都做,受美國政策扶持(本土製造、補貼)。

  • 優點: 唯一美系、地緣上受扶持、技術不落後。
  • 缺點: 規模小於兩韓廠,曾受中國市場管制衝擊。

鎧俠 Kioxia(日,IDM)

原東芝記憶體獨立而來,NAND Flash 的發明者,專注 NAND,與 Sandisk 長期合資生產。

  • 優點: NAND 老牌技術深、日系品質,專注度高。
  • 缺點: 不做 DRAM、產品線單一,財務與週期波動較敏感。

台廠群像(南亞科 / 華邦 / 旺宏 / 群聯 / 慧榮)

南亞科做利基 DRAM,華邦橫跨 NOR 與利基 DRAM,旺宏強在 NOR 與 ROM;群聯、慧榮則是全球數一數二的 NAND 主控廠。

  • 優點: 利基、NOR、主控、模組這些縫隙站得穩,彈性與韌體能力強。
  • 缺點: 規模與先進製程顆粒打不過國際 IDM,易受大廠價格策略牽動。

中國廠(長江存儲 YMTC / 長鑫 CXMT)

長江存儲做 NAND、長鑫做 DRAM,以國家資本切入,技術追趕中,是未來幾年最大變數——也是出口管制的重點對象。

  • 優點: 國家補貼、內需龐大、追趕速度快。
  • 缺點: 先進製程與 HBM 受管制卡關,生態與良率仍落後,帶地緣風險。

待查證:HBM 領先態勢、中國廠進度、各家市占與優缺點為市場通論與定位歸納,會隨景氣與政策變動,引用前宜更新來源。
