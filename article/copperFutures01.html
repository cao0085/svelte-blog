找礦

礦業風險最高、最慢的部分就在「動鏟之前」。一座礦從找到到投產，大致要走過這幾關，通常 5–15 年：探勘（圈出可能有礦的區域）→ 鑽探取岩心化驗 → 估算資源量與儲量 → 可行性研究（算出在什麼銅價下賺錢）→ 拿許可、籌資、建廠才能投產。

礦場評估的是這整片礦石裡所有有價金屬值不值得挖，常見的共生組合：

  • 斑岩銅礦（porphyry copper）：銅 + 鉬 + 金 + 銀，全球大部分銅來自這類（智利、秘魯的大礦）
  • 銅金礦：銅 + 大量金（如印尼 Grasberg，同時是全球數一數二的金礦）
  • 沉積型銅礦：銅 + 鈷 + 銀（如剛果銅帶的 Kamoa-Kakula）
  • VMS / 多金屬礦：銅 + 鋅 + 鉛 + 金銀（如 Hudbay、Boliden 那類）

找礦有兩條路。Junior Explorer 是「風險承擔者 + 找礦專家」，本身持有礦權，找到礦之後把資產賣給大集團或接受入股。大集團（BHP、Rio Tinto）則多數自己養勘探部門，在全球幾十個國家同時佈局，直接跟政府談礦權，也會主動收購已找到礦的 Junior 公司。

生產成本

業界把成本分三層：C1 現金成本（採礦 + 選礦 + 冶煉，扣掉副產品收入）、AISC 全維持成本（C1 + 資本支出 + 權利金）、All-in Cost（再加新建礦資本）。看財報的「現金成本 1.91 美元/磅」指的是 C1，而且裡面已經混了副產品（金/銀/鉬）的抵減。

把所有礦場按 C1 由低到高排出來就是成本曲線，銅價像一條水平線切過去——線以下的礦場賺錢，線以上的虧錢可能停產。這條曲線解釋了幾件事：

  • 低成本礦站在曲線左側，銅價漲利潤幾乎全進口袋；銅價跌，它們最後一個倒下
  • 高成本礦在低價時減產，是「銅價下跌自動修正供給」的機制
  • 副產品大漲時，整條曲線整體下移，即使銅價沒漲，礦商財報也會很漂亮

礦業必然先挖品位最高的部分，所以礦場越老平均品位自然下滑，全球銅礦品位已從過去的 1–2% 掉到 0.7% 以下。同樣要產出 1 噸銅，現在得挖比十年前更多的土石，成本、能耗、水需求全部上升，在缺水的智利、環評趨嚴的地區擴產越來越難。帳面儲量雖然還在增加，但那些新增儲量大多是更低品位、更深、更難挖的礦體 —— 全球不是「沒有銅」，而是「好挖的銅越來越少」。

售價

絕大多數金屬實物合約（礦場賣給冶煉廠、貿易商賣給製造商）都不訂死價格，而是以 LME 官方結算價為基準加減升貼水成交價 = LME 均價（某定價期）± 升貼水，所以礦場是標準的價格接受者（Price Taker），LME 是價格發現（Price Discovery）的場所，礦場唯一能「主動定價」的工具，就是去 LME 賣出期貨對沖鎖住未來價格。

宏觀預期 + 投機資金
        ↓
    LME 期貨價格   ← 真正的定價中心
        ↓
  實物現貨跟隨
        ↓
  礦場被動接受

有趣的是，這邊的供需法則參考點已經改變；不是看實物買賣，而是期貨市場對未來供需的「預期」，可能造成倉庫裡銅堆積如山，帳面售價卻還是高的現象。

市場預期需求回升 → LME 期貨上漲 → 現貨跟漲（即使倉庫堆積如山）→ 採購商看到高價觀望不買 → 礦場帳面售價高、實物卻滯銷

開採週期

礦場的時間尺度非常長，從勘探到第一噸礦出來，通常要 10–20 年。建礦(環評、社區談判、基礎建設)本身就要 5–10 年，與政府的礦權合約（特許經營協議）通常簽 25–50 年。合約裡訂的不是金屬售價，而是：

  • 權利金（Royalty）：每賣出一噸抽幾 %，通常掛鉤 LME 價格浮動
  • 稅率結構：企業所得稅、超額利潤稅
  • 最低開採量義務

通膨

礦權部分政府拿的是「浮動抽成」，所以通膨對這塊影響不大。受通膨影響的是礦場成本端：勞工薪資、能源、炸藥、設備零件。一旦通膨升溫 C1 成本就往上跑，但售價不一定能同步漲，利潤空間被壓縮。

停產

上面有提到礦場是價格接受者，所以能控制的是「要不要在這個價格下繼續生產」，停產就成了一種風險控管的手段，有分成三種:

  • 減產（Curtailment）: 減少班次、暫停部分採區、降低選礦廠處理量。設備還在轉人還在，幾週內可以回滿產。

  • 維護性停產（Care & Maintenance）: 礦場停止生產，但保留最低限度的人員維持設備不損壞、排水系統繼續運轉、安全監控維持。地下礦特別需要這個，因為一旦停止抽水，坑道會淹水重啟成本會暴增。這個狀態可以維持數年，重啟需要 6–18 個月。

  • 永久關閉（Closure）: 這是不可逆的選擇包含設備拆除、坑道封填、環境復原等等。

實物貿易商

核心角色是中間流動性提供者(類似果農中盤商的概念)，它向礦場買精礦、安排物流船運倉儲、再賣給下游冶煉廠或製造商。LME 對他們是避險工具，報價格式固定是 LME 某月期貨價 ± 升貼水，：

買進實物（付 LME + 升貼水 A）
        ↓
同時在 LME 賣出等量期貨（鎖住價格風險）
        ↓
賣出實物給下游（收 LME + 升貼水 B）
        ↓
獲利 = 升貼水 B − 升貼水 A（與 LME 漲跌無關）

實物漲價損失 → 期貨空單獲利 對沖
實物跌價獲利 → 期貨空單損失 對沖

所以淨損益理論上鎖死在升貼水價差，不受市場方向影響，而實務上有幾個風險：

  • 升貼水變動：買進時升貼水是 +$80，找到下游買家時可能已壓縮到 +$50，這段差價是實質虧損，且沒有對沖工具可以覆蓋。
  • 基差風險（Basis Risk）：LME 對沖的是「標準品銅」，實物可能是特定礦場、特定品位的精礦，對沖永遠不是 100% 精準。
  • 實物損耗與品質風險：金屬在倉儲、運輸過程中有損耗、污染、品質劣化的可能，這些無法用期貨對沖。
  • 流動性風險：庫存壓太久，資金被佔用、融資成本上升。若市場突然凍結，被迫在差的價格出清。

市場

LME 最活躍的合約是三個月期貨，所以銅價主要反映當下實物供需加上未來 3–6 個月的預期；超過半年的故事會影響市場情緒，但套利機制會把過度偏離的期貨拉回來，這個「到期錨定實物」的邏輯適用於所有實物商品期貨，但銅有幾個地方特別突出：

  • 流動性特別深：LME、上海期交所、COMEX 三所同時有活躍合約，價格發現效率高，難被單一勢力操控。
  • 被當作經濟領先指標：銅用在幾乎所有工業環節，市場習慣把銅價當作全球經濟健康度的代理指標，金融化程度比其他基本金屬高。
  • 中國集中度極高：全球銅消費約 55% 在中國，中國政策和經濟數據對銅價的影響特別直接。

不過短期仍然可以被金融資金扭曲 —— 2020–2021 年銅價從 $5,000 漲到 $10,000，實物需求沒有翻倍，很大一部分是美元流動性氾濫加上基金押注綠能的資金行情。這種扭曲通常會被現實修正：庫存開始累積、TC 上升、現貨升水消失然後價格回落。股票沒有這種自動修正機制，銅有。所以銅的絕對價格是落後指標，若要分辨「真實供需還是資金行情」，可以參考幾個點:

TC/RC 走勢

每年礦場和冶煉廠談判的加工費，是領先 LME 的早期訊號：

TC 高  =  精礦供給過剩，礦場多、冶煉廠少
          → 冶煉廠可以挑，開高價
          → 產能利用率高，銅產出高

TC 低  =  精礦供給緊張，礦場少、冶煉廠多
          → 冶煉廠搶不到原料，被迫壓低要求
          → 產能利用率低，銅產出少

銅價結構

LME 現貨價 vs 三個月期貨的價差比絕對價格更能反映實物鬆緊：

  • 現貨貴過期貨（Backwardation）→ 實物緊張，市場現在缺貨
  • 期貨貴過現貨（Contango）→ 供給寬鬆，庫存充裕

實際消費量

實物銷售量難以直接觀察，替代指標：

  • LME + 上海期交所 + COMEX 公開倉庫庫存變化
  • 中國銅進口量（中國佔全球銅消費約 55%，最關鍵）
  • 精煉銅表觀消費量（產量 + 進口 − 出口 ± 庫存變化）

礦場供給端

  • 產量指引（Production Guidance）：礦業集團每季財報公布，偏離指引就是訊號
  • C1 現金成本：判斷當前銅價下有多少礦場在虧損邊緣，決定潛在停產量
  • 礦場停產 / 復產公告：直接影響未來供給
TC/RC 走勢       → 最領先，反映礦端供給
        ↓
LME 庫存 + 價差  → 反映當下實物鬆緊
        ↓
中國進口量       → 反映真實消費需求
        ↓
礦場產量指引偏離 → 確認供給端變化
        ↓
銅價絕對價格     → 最滯後，是前面所有因素的結果
